## 必要条件
コードを実行するには，下記のソフトウェアがインストールされている必要があります．

* Python 3.x
* NumPy
* Matplotlib
* その他，必要に応じて各種パッケージを利用

※Pythonのバージョンは，3系を利用します．


## 実行方法

ファイルを展開したディレクトリへ移動し，jupyter-notebook を実行します．

```
$ jupyter-notebook
```

jupyter-notebook用のコード(ch01intro.ipynbなど)をダブルクリックして実行します．
